# Snap Translate

> 📖 **언어**: [English](./README.md) · **한국어**

macOS와 Windows용 키보드 중심의 가벼운 번역 도구. 어떤 앱에서든 텍스트를 선택하고 글로벌 단축키 한 번이면 DeepL 번역 결과가 클립보드에 즉시 복사됩니다. 지금 하던 작업을 멈추지 않고 그대로 이어갈 수 있습니다.

## 왜 Snap Translate인가?

대부분의 번역 도구는 별도의 앱이나 브라우저 확장으로 만들어져 있어, 번역이 필요할 때마다 **작업 맥락을 끊어야 합니다**. 텍스트를 복사하고, 새 창을 열고, 붙여넣고, 번역 버튼을 누르고, 결과를 읽은 뒤 다시 원래 창으로 돌아가는 일련의 과정은 사소해 보이지만 하루에 수십 번 반복되면 집중력과 생산성을 크게 갉아먹습니다. 문서를 읽을 때, 이메일을 확인할 때, 모르는 언어로 된 코드 주석을 마주할 때마다 이 마찰이 누적됩니다.

**Snap Translate는 바로 그 마찰을 완전히 없애기 위해 만들어졌습니다.**

### 설계 목표

1. **맥락 전환 없음** — 이 도구는 시스템 트레이에 상주합니다. 설정을 바꿀 때가 아니라면 새 창이 뜨지 않습니다.
2. **한 번의 단축키로 끝** — 텍스트 선택 → 단축키 한 번 → 클립보드에 번역 완료. 이게 전부입니다.
3. **어디서든 작동** — 포커스된 앱에 `Cmd+C` / `Ctrl+C`를 시뮬레이션하는 방식이므로 네이티브 앱, 브라우저, 터미널, IDE 등 어디서든 동작합니다. 앱별 연동이 필요 없습니다.
4. **클립보드를 존중** — 번역 전에 현재 클립보드 내용을 백업하고, 번역이 끝나면 자동으로 복원합니다. 복사해둔 내용이 조용히 날아가는 일이 없습니다.
5. **네이티브 성능, 작은 용량** — Tauri 2 + Rust 백엔드 기반입니다. 동급 Electron 앱 대비 메모리 점유가 훨씬 적고, 시작이 즉각적입니다.
6. **기본값이 곧 안전한 값** — DeepL API 키는 OS 키체인(macOS Keychain / Windows Credential Manager)에만 저장되며, 설정 파일에 평문으로 남지 않습니다.

### 누구를 위한 도구인가?

- 영문 기술 문서, 스택 트레이스, GitHub 이슈를 자주 읽는 개발자
- 외국어 논문과 자료를 훑어봐야 하는 연구자
- 여러 언어로 된 이메일·기사·메시지를 매일 여러 번 접하는 사람
- 무거운 GUI보다 **키보드 중심의 집중 흐름**을 선호하는 사용자

### 왜 DeepL인가?

DeepL은 특히 유럽과 동아시아 언어에서 경쟁 서비스 대비 훨씬 자연스럽고 문맥을 잘 반영하는 번역을 제공합니다. 무료 티어도 월 50만 자로 넉넉한 편이어서 개인 사용자라면 대부분 커버됩니다. Snap Translate는 API 키 접미사에 `:fx`가 있는지를 보고 무료/유료를 자동 감지해 알맞은 엔드포인트로 요청을 보냅니다.

## 주요 기능

- 🖱️ **시스템 트레이 상주형** — 화면 공간을 차지하는 창이 없습니다.
- ⌨️ **글로벌 단축키** — `Cmd+Shift+D` (macOS) / `Ctrl+Shift+D` (Windows)
- 🌐 **DeepL 기반 번역** — 원본 언어 자동 감지와 함께 고품질 번역 제공
- 🔁 **자동 재시도** — 요청 속도 제한 발생 시 1s → 2s → 4s → 8s 지수 백오프
- 📋 **클립보드 백업·복원** — 기존 클립보드 내용을 절대 잃지 않음
- 🔐 **안전한 API 키 저장** — OS 네이티브 키체인 사용, 디스크에 평문 저장 안 함
- 🌍 **9개 대상 언어** — 한국어, 영어, 일본어, 중국어 간체, 독일어, 프랑스어, 스페인어, 포르투갈어(브라질), 러시아어
- 🔔 **데스크톱 알림** — 창을 열지 않고도 번역 결과 미리보기 확인
- 📝 **일별 롤링 로그** — 디버깅용 로그 파일 자동 관리
- 🛡️ **동시 실행 방지** — 단축키를 연타해도 중복 번역이 실행되지 않음

## 시작하기

### 1. DeepL API 키 발급

1. [DeepL API](https://www.deepl.com/pro-api)에 가입합니다.
2. 계정 페이지에서 인증 키를 복사합니다.
3. 접미사가 `:fx`인 키는 무료 티어이며 자동으로 `api-free.deepl.com`으로, 접미사가 없는 키는 `api.deepl.com`으로 라우팅됩니다.

### 2. Snap Translate 실행

- 첫 실행 시 메뉴바(macOS) 또는 시스템 트레이(Windows)에 아이콘이 나타납니다.
- **macOS 사용자**: **Accessibility** 권한을 요청하는 시스템 대화상자가 나타납니다. 선택된 텍스트에 대해 복사 단축키를 시뮬레이션하려면 이 권한이 필요합니다. *시스템 설정 → 개인정보 보호 및 보안 → 손쉬운 사용*에서 허용해 주세요.

### 3. API 키 등록

1. 트레이 아이콘을 클릭하거나 우클릭 → **설정**을 선택합니다.
2. DeepL API 키를 붙여넣고 **저장**을 누릅니다. 앱은 저장 전에 DeepL의 `/v2/usage` 엔드포인트로 키 유효성을 검증합니다.
3. 드롭다운에서 선호하는 대상 언어를 선택합니다.

### 4. 바로 번역하기

- 어디서든 텍스트를 선택합니다 — 브라우저, Slack, PDF 뷰어, 터미널 등.
- **`Cmd+Shift+D`** (macOS) 또는 **`Ctrl+Shift+D`** (Windows)를 누릅니다.
- 번역 결과의 첫 100자가 데스크톱 알림으로 표시됩니다.
- 전체 번역본은 이미 클립보드에 있습니다. 원하는 곳에 붙여넣으세요.

### 내부 동작 원리

단축키를 누르면 Snap Translate는 다음 단계를 원자적으로 실행합니다:

1. **동시 실행 가드** — 이미 번역이 진행 중이라면 새 요청은 무시됩니다.
2. **Accessibility 확인** (macOS) — 권한이 여전히 있는지 확인하고, 해지되었다면 다시 요청합니다.
3. **API 키 확인** — OS 키체인에서 키를 조회합니다. 없다면 설정 창을 엽니다.
4. **클립보드 백업** — 현재 클립보드 내용(텍스트 또는 이미지)을 저장합니다.
5. **복사 시뮬레이션** — 포커스된 앱에 `Cmd+C` / `Ctrl+C`를 보냅니다.
6. **재시도 읽기** — 50/100/200/400/800ms 간격으로 새로운 클립보드 내용을 탐지합니다.
7. **길이 검사** — 5,000자를 초과하면 거부합니다 (DeepL 요청당 한도).
8. **번역** — DeepL `/v2/translate`를 호출하며 429 Rate Limit에 대해서는 지수 백오프로 재시도합니다.
9. **복원 & 교체** — 먼저 원본 클립보드를 복원하고, 그 뒤 번역 결과를 기록합니다.
10. **알림** — 데스크톱 알림으로 결과를 표시합니다.

어디선가 실패하면 어떤 단계가 왜 실패했는지(네트워크 오류, 유효하지 않은 키, 할당량 초과, 선택된 텍스트 없음, 텍스트 너무 긺 등)를 한국어로 명확히 알려주며, 원본 클립보드는 항상 복원됩니다.

## 설정

설정은 성격에 따라 두 곳에 나누어 저장됩니다:

| 설정 | 저장 위치 | 이유 |
|---|---|---|
| **DeepL API 키** | OS 키체인 (`service="snap-translate"`, `username="deepl-api-key"`) | 비밀 정보 — 디스크에 평문으로 두면 안 됨 |
| **대상 언어, 단축키** | 앱 설정 디렉토리의 `settings.json` | 비밀이 아닌 일반 환경설정 |

**설정 디렉토리 위치:**
- macOS: `~/Library/Application Support/SnapTranslate/`
- Windows: `%APPDATA%\SnapTranslate\`

**로그 디렉토리 위치** (`snap-translate.log.YYYY-MM-DD` 형식의 일별 롤링 파일):
- macOS: `~/Library/Logs/SnapTranslate/`
- Windows: `%APPDATA%\SnapTranslate\logs\`

## 문제 해결

### "선택된 텍스트가 없습니다"

- macOS라면 *시스템 설정 → 개인정보 보호 및 보안 → 손쉬운 사용*에서 Snap Translate에 권한이 부여되어 있는지 확인하세요. 권한이 없으면 OS가 시뮬레이션된 `Cmd+C`를 조용히 차단합니다.
- 텍스트가 실제로 선택(하이라이트)되어 있는지 확인하세요.
- 일부 앱(잠긴 PDF 뷰어, DRM이 걸린 콘텐츠 등)은 복사 자체를 막으므로 번역할 수 없습니다.

### "API Key가 유효하지 않습니다"

- 붙여넣은 키를 다시 확인하세요 — 무료 티어 키는 DeepL이 제공하는 그대로 `:fx` 접미사가 포함되어 있어야 합니다.
- [DeepL API 대시보드](https://www.deepl.com/pro-account)에서 계정이 정상 상태인지 확인하세요.

### "번역 할당량이 초과되었습니다"

- 이번 달 DeepL 글자 할당량을 모두 썼습니다. 무료 티어는 월 50만 자, 유료는 요금제에 따라 다릅니다.
- 초기화 날짜는 DeepL 대시보드에서 확인할 수 있습니다.

### 알림이 뜨지 않는다

- macOS: *시스템 설정 → 알림*에서 Snap Translate의 알림 권한을 허용하세요.
- Windows: *설정 → 시스템 → 알림*을 확인하세요.

### 로그

트레이 메뉴의 **로그 열기**로 로그 디렉토리를 연 뒤, 이슈 제보 시 최신 일별 로그를 첨부해 주세요.

## 아키텍처

Snap Translate는 **Rust 백엔드**(Tauri)와 설정 UI 전용 **React 프론트엔드**로 깔끔하게 분리되어 있습니다.

```
┌─────────────────────────────────────────────────┐
│                   시스템 트레이                 │
│   [설정]  [로그 열기]  [종료]                   │
└─────────────┬───────────────────────────────────┘
              │
              ▼
┌─────────────────────────────────────────────────┐
│           React 19 설정 창                      │
│  ApiKeyInput · LanguageSelect · HotkeyInput     │
│       ↕ invoke("command_name", args)            │
└─────────────┬───────────────────────────────────┘
              │
              ▼
┌─────────────────────────────────────────────────┐
│           Tauri 2 Rust 백엔드                   │
│                                                 │
│  글로벌 단축키 ──► translate_flow               │
│     │                 │                         │
│     ▼                 ▼                         │
│  [Accessibility]  [Clipboard]  [KeySim]         │
│  [ApiKeyStore]    [DeepL HTTP] [Notification]   │
└─────────────────────────────────────────────────┘
              │
              ▼
     DeepL API (무료/유료 자동 라우팅)
```

### 핵심 설계 패턴

- **Trait 기반 의존성 주입** — 모든 외부 I/O(클립보드, HTTP, 키체인, 키 시뮬레이션, accessibility)를 trait 뒤에 숨깁니다. 덕분에 mock으로 전 구간 단위 테스트가 가능하며 번역 플로가 순수하고 검증 가능합니다.
- **RAII 가드** — `TranslationGuard`가 `AtomicBool`을 사용해 동시 번역을 막습니다. 패닉이 발생해도 `Drop`에서 플래그가 해제됩니다.
- **타입이 있는 에러** — `AppError`(thiserror 기반)가 사용자용 한국어 메시지를 담고 있으며, Tauri 경계를 넘을 때 평문 문자열로 직렬화됩니다.
- **지수 백오프 재시도** — `translate_with_retry`는 `429 Rate Limited`에 대해서만 1/2/4/8s 지연으로 재시도하고, 다른 오류는 즉시 실패합니다.

더 깊은 문서는 저장소 전반에 있는 계층화된 [`AGENTS.md`](./AGENTS.md) 파일들을 참고하세요.

## 보안과 개인정보

- **DeepL API 키는 이 기기를 벗어나지 않습니다.** 오직 DeepL로 직접 HTTPS 요청을 보낼 때 헤더로만 사용됩니다.
- **텔레메트리, 분석, 추적 없음.** 이 앱은 DeepL로만 네트워크 호출을 합니다.
- **클립보드 로깅 없음.** 클립보드 내용은 단일 번역 요청이 진행되는 동안만 메모리에 존재합니다.
- **로그는 운영 메타데이터만 담습니다** (타임스탬프, 에러, 재시도 횟수 등). 번역 내용이나 API 키는 로그에 남지 않습니다.
- API 키는 `keyring` 크레이트의 `apple-native` (macOS Keychain) / `windows-native` (Windows Credential Manager) 백엔드로 저장됩니다.
